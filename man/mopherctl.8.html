NAME

mopherctl — the mopher daemon control utility

SYNOPSIS

mopherctl [−h] [−d level] [−c file] [−s host] command

DESCRIPTION

mopherctl controls the mopher daemon.

Available options are:

−h

Show usage message.

−d level

Set logging severity level to level

−c file

Read configuration from file.

−s host

Connect to host.

command is one of the following:

dump table

Print raw content of table.

greylist dump

Print formatted content of the greylist-table.

greylist pass origin from rcpt

Temporarily whitelist triplet.

FILES
@sysconfdir@/mopher/mopherd.conf

Configuration file for the mopher daemon.

@sysconfdir@/mopher/mopherd.acl

List of rules enforced by the mopher daemon.
