NAME

mopherd — the mopher daemon

SYNOPSIS

mopherd [−hfC] [−d level] [−c file] [−p pidfile]

DESCRIPTION

mopherd reads a list of rules and acts on each incoming mail delivery attempt based on various criteria and existing states.

Available options are:

−h

Show usage message.

−f

Run in foreground, don’t detach from the controlling terminal.

−C

Check configuration file syntax and exit.

−d level

Set logging severity level to level.

−c file

Read configuration from file.

−p pidfile

Create a PID file in pidfile.

FILES
@sysconfdir@/mopher/mopherd.conf

Configuration file for the mopher daemon.

@sysconfdir@/mopher/mopherd.acl

List of rules enforced by the mopher daemon.
